## ðŸ”¹ Setup & Configuration

### git --version
Checks installed Git version.
Example:
git --version

### git config --global user.name "Your Name"
Sets your Git username.
Example:
git config --global user.name "Satyam"

### git config --global user.email "you@example.com"
Sets your Git email.
Example:
git config --global user.email "satyam@email.com"

### git config --list
Shows current Git configuration.
Example:
git config --list

## ðŸ”¹ Repository Basics

### git init
Initializes a new Git repository.
Example:
git init

### git status
Shows current state of working directory.
Example:
git status

### git add file
Stages file for commit.
Example:
git add git-commands.md

### git add .
Stages all changes.
Example:
git add .

### git commit -m "message"
Creates a commit with message.
Example:
git commit -m "Added initial git commands"

---

## ðŸ”¹ Viewing History

### git log
Shows detailed commit history.
Example:
git log

### git log --oneline
Shows compact commit history.
Example:
git log --oneline

### git diff
Shows changes not yet staged.
Example:
git diff

### git diff --staged
Shows staged changes.
Example:
git diff --staged
